# ✨ IkeHub

> 🌌 A premium interactive Three.js portal hub for the **Ikeverse** ecosystem.

**IkeHub** showcases **The Culturalverse**, **The Living Knowledge Platform**, **IkeStar**, and **The Cosmic Weave** through a cinematic 3D experience with responsive UI, cosmic visuals, and immersive navigation.

---

## 🚀 Features

- 🌌 Interactive **3D hub** built with **Three.js**
- 🪐 Central **IkeHub core** with connected app portals
- 🌍 Portal nodes for:
  - **The Culturalverse**
  - **The Living Knowledge Platform**
  - **IkeStar**
  - **The Cosmic Weave**
- ✨ Cinematic cosmic background and atmospheric effects
- 📱 Responsive layout for **desktop, tablet, and mobile**
- 🎯 Focused app selection with smooth camera transitions
- 🧭 Portal labels, app cards, and dock navigation
- 🧩 Modular app configuration for easy expansion

---

## 🛠 Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **Three.js**
- **Three.js Postprocessing**
